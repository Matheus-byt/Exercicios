TAGS HTML

<strong> Deixar em negrito </strong> ANTIGA TAG = <b> bold
<em> Dar ênfase </em> 
<mark> Marcas o texto </mark>
<del> Riscar texto </del>
<ins>Deve ler o texto</ins>
<sub> Deixar numero pequeno debaixo da letra
<sup> Deixar número pequeno em cima da letra

<abbr> - ABREVIAÇÃO

<pre><code> SERVE PARA DIGITAR CÓDIGO </code></pre>

<q> Coloca aspas automaticamente para citar informações </q>

<blockquote> Serve para dar espaçamento automaticamente</blockquote>

TAGS CSS

background-image: linear-gradient(to right, #color, #color) - Colocar degrade;
font-size: sempre utilizar em ou px (Define tamanho da fonte) 1em = 16px (tamanho padrão).
font-family: Mudar a fonte;
font-weight: (lighter normal bold bolder) ou de 100 a 900;
font-style: italic (define o estilo da fonte);
text-decoration: underline; (Coloca o sublinhado);

SHORTHAND FONT (Define os estilos de maneira curta sem precisar de tantas linhas).
